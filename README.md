## Installation

LZCommonSDK is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'LZCommonSDK, :git => 'https://github.com/rm-c/LZCommonSDK.git''
```
